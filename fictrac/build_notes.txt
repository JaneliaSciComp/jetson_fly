Install basic requirements
################################

* git  (sudo apt-get install git)

* cmake (sudo apt-get install cmake)

* g++ (sudo apt-get install g++)

* libopencv-dev (sudo apt-get install libopencv-dev

* libnlopt-dev (sudo  apt-get install libnlopt-dev)

* libcairomm-1.0  (sudo apt-get install libcairomm-1.0-dev)

* libboost-dev (sudo apt-get install libboost-dev)

* checkinstall (sudo apt-get install checkinstall)

Install cpp_redis and dependencies
##################################

# Clone the project
git clone https://github.com/Cylix/cpp_redis.git

# Go inside the project directory
cd cpp_redis

# Get tacopie submodule
git submodule init && git submodule update

# Create a build directory and move into it
mkdir build && cd build

# Generate the Makefile using CMake
cmake .. -DCMAKE_BUILD_TYPE=Release

# Build the library
make

# Install the library
sudo checkinstall   (answer questions etc).
